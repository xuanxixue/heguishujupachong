## 📖 项目概述

**AI数据采集浏览器**是一款集成了现代网页浏览、智能数据爬取、多语言翻译和多种格式导出的综合性工具。该软件专为AI训练数据收集、学术研究、内容分析等场景设计，提供了直观的图形界面和强大的数据处理能力。

## ✨ 核心功能

### 🌐 现代化浏览器
- 完整的网页浏览体验（前进、后退、刷新）
- 地址栏智能补全和导航
- 基于Chromium内核，支持现代Web标准
- 响应式界面设计，支持高DPI显示

### 🕷️ 智能数据爬取
- **智能检测**：自动检查网站可爬取性，遵守robots.txt规则
- **双重爬取引擎**：
  - Requests库：快速爬取静态页面
  - Selenium：处理JavaScript渲染的动态页面
- **内容提取**：智能识别和提取文本内容，过滤无关元素
- **合法性保障**：内置爬虫礼仪，避免对服务器造成负担

### 🔤 多语言翻译
- 集成Google翻译API
- 支持中英文互译
- 实时翻译爬取内容
- 异步翻译处理，不阻塞界面

### 💾 多格式导出
- **JSON格式**：结构化数据，便于程序处理
- **纯文本格式**：专为AI训练优化的格式
- **DOCX文档**：专业的Word文档格式，包含完整排版
- **自动保存**：爬取数据实时保存，防止丢失

### 🎨 现代化界面
- 清新现代的UI设计
- 直观的选项卡布局
- 实时状态反馈和进度显示
- 可调整的分割面板

## 🛠️ 技术架构

### 核心技术栈
- **前端界面**：PyQt5 + QtWebEngine
- **爬虫引擎**：Requests + BeautifulSoup4 + Selenium
- **翻译服务**：Google Translate API
- **文档处理**：python-docx
- **浏览器驱动**：ChromeDriver

### 系统要求
- **操作系统**：Windows 10/11, Linux, macOS
- **Python版本**：3.11+
- **内存**：4GB RAM（推荐8GB）
- **存储空间**：500MB可用空间

## 🚀 使用说明

### 基本操作流程

1. **启动程序**
   - 运行`python liulanqi.py`
   - 程序自动初始化浏览器和爬虫组件

2. **浏览网页**
   - 在地址栏输入网址，按回车访问
   - 使用导航按钮进行前进、后退、刷新操作

3. **检查可爬取性**
   - 点击"检查可爬取性"按钮
   - 系统自动分析当前页面是否允许爬取

4. **爬取数据**
   - 点击"爬取页面"开始数据提取
   - 实时显示爬取状态和进度

5. **查看和管理数据**
   - 在右侧面板查看已爬取的数据列表
   - 点击数据项查看详细内容
   - 使用翻译功能处理多语言内容

6. **导出数据**
   - 选择适合的导出格式：
     - **保存数据**：JSON格式
     - **导出训练数据**：纯文本格式
     - **导出DOCX**：Word文档格式

### 高级功能

#### 批量爬取
虽然界面主要针对单页爬取，但代码架构支持批量处理。有经验的用户可以修改代码实现自动化批量采集。

#### 自定义爬取规则
通过修改`extract_text_content`方法，可以自定义内容提取规则，适应不同的网站结构。

#### 翻译配置
支持修改翻译目标语言，默认支持中英文互译。

## 🔧 故障排除

### 常见问题

#### 1. Selenium初始化失败
**问题**：无法找到Chrome浏览器
**解决方案**：
- 确认已安装Google Chrome浏览器
- 手动设置Chrome安装路径
- 或使用系统包管理器安装Chrome

#### 2. 翻译功能不可用
**问题**：网络连接问题或API限制
**解决方案**：
- 检查网络连接
- 确认能访问Google翻译服务
- 考虑使用本地翻译替代方案

#### 3. 内存占用过高
**问题**：处理大量页面时内存使用增加
**解决方案**：
- 定期清空已爬取数据
- 增加系统虚拟内存
- 分批处理大量数据

#### 4. 某些网站无法爬取
**问题**：网站有反爬虫机制
**解决方案**：
- 调整请求头信息
- 增加请求延迟
- 使用Selenium模拟真实用户行为

## 📊 数据格式说明

### JSON格式示例
```json
{
  "url": "https://example.com",
  "title": "示例页面",
  "content": "提取的文本内容...",
  "length": 1500,
  "crawl_time": "2024-01-01 12:00:00"
}
```

### 训练数据格式
```
URL: https://example.com
标题: 示例页面
内容: 提取的文本内容...

================================================================================

URL: https://example2.com
标题: 另一个页面
内容: 更多文本内容...
```

### DOCX文档结构
- 封面标题：网页数据采集报告
- 采集元数据：时间、页面数量
- 按页面组织的结构化内容
- 专业的段落和标题格式
